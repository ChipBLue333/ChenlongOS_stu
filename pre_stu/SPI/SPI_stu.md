# SPI 协议深度解析手册

> **目标**：新手快速掌握 SPI 通信标准。

## 1. 协议核心
* **同步**：共用 SCLK 时钟。
* **全双工**：主从机同时双向收发。
* **主从结构**：主机发起，产生时钟。

## 2. 硬件接口
| 线名 | 全称 | 方向 |
| :--- | :--- | :--- |
| **SCLK** | 时钟 | 主 -> 从 |
| **MOSI** | 主发从收 | 主 -> 从 |
| **MISO** | 主收从发 | 从 -> 主 |
| **CS** | 片选(低有效)| 主 -> 从 |

## 3. 四种模式 (CPOL/CPHA)
* **Mode 0**: 空闲低, 上升沿采样。
* **Mode 1**: 空闲低, 下降沿采样。
* **Mode 2**: 空闲高, 下降沿采样。
* **Mode 3**: 空闲高, 上升沿采样。

## 4. 软件思路
1. 拉低 **CS**。
2. 调用传输函数（发 1 字节并收 1 字节）。
3. 拉高 **CS**。
*注意：读数据需发送 Dummy Byte (如 0xFF)。*

## 5. 避雷点
* CS 事务完整性。
* SCLK 频率匹配。
* MSB First 默认位序。
